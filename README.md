# Password Strength Analysis and Login Security Monitoring System

## Overview

A cybersecurity-focused web application developed using Python Flask, SQLite, HTML, CSS, and JavaScript. The system analyzes password strength in real time and monitors login attempts to detect suspicious activity.

## Features

* Real-time password strength analysis
* Password risk classification
* Login attempt monitoring
* Multiple failed login detection
* Security alert generation
* SQLite database integration
* Security analytics dashboard
* CSV export functionality

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript
* Kali Linux

## Project Objectives

* Improve password security awareness
* Detect weak passwords
* Monitor authentication attempts
* Generate alerts for suspicious behavior
* Demonstrate basic security monitoring concepts

## Future Enhancements

* Multi-Factor Authentication (MFA)
* Password hashing
* Email alerts
* SIEM integration
* Machine learning-based anomaly detection
